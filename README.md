# KZ Guide MEGA PT-BR

Pack completo de guias para o addon **KZ Guide** — WoW 1.12.1 (Vanilla) em **Portugues Brasileiro**.

Desenvolvido para o servidor **SandWorlds** (interface 11200 / build 5875).

---

## O que esta incluido

| Pack | Conteudo |
|---|---|
| `KZ_Guide` | Addon principal (engine, parser, UI, banco de dados) |
| `KZ_Guide_Alliance` | Guia de leveling Alliance 1-60 PT-BR |
| `KZ_Guide_Horde_PTBR` | Guia de leveling Horde 1-60 PT-BR |
| `KZ_Guide_Dungeons_PTBR` | Guias de masmorras PT-BR |
| `KZ_Guide_Professions_PTBR` | Profissoes 1-300 (Alchemy, Blacksmithing, Cooking, Enchanting, Engineering, First Aid, Fishing, Herbalism, Leatherworking, Mining, Skinning, Tailoring e mais) |
| `KZ_Guide_Rogue_PTBR` | Habilidades de Ladino — Arrombamento (Lockpicking) 1-300 |
| `KZ_Guide_LV60_PTBR` | Conteudo de nivel 60 / endgame |

---

## Como instalar

1. Baixe ou clone este repositorio
2. Copie as pastas desejadas para:
```
WoW\Interface\AddOns\
```
3. O addon principal `KZ_Guide` e **obrigatorio** — os packs dependem dele
4. Inicie o jogo e ative os addons na tela de selecao de personagem
5. No jogo: `/kz show` para abrir o guia

---

## Requisitos

- World of Warcraft **1.12.1** (cliente Vanilla)
- Servidor compativel com interface **11200**
- Testado no servidor **SandWorlds**

---

## Comandos no jogo

```
/kz show       — abrir o guia
/kz hide       — fechar o guia
/kz settings   — configuracoes
/reload        — recarregar addons apos atualizacao
```
